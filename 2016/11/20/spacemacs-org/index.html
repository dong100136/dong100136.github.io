spacemacs使用小记

Nov 20, 2016 | Hits

写这篇博文主要是为了记录一些spacemacs的一些快捷键和一些使用方法，用于自己速查。

base

按键 说明
SPC b m kill other buffer
SPC b b show buffer list
SPC b s goto scratch
SPC b d kill this buffer

ssh

最近发现spacemacs一个超级棒的功能，在C-x C-f中输入url就可以使用tramp这个插件来访问远端的文件，可以像在本机一样进行操作，非常棒
url的格式如下/:@: protocal一般为ssh

org-todo

在普通状态下:

按键 说明
t 讲项目设置为TODO项目或者在TODO，DONE，NONE之间进行切换
S-UP/DOWN 同上
, : 创建标签

VCS

按键 说明
SPC g i git init
SPC g s git status
c c git commit
s status a file
S status all files

vim操作

按键 说明
S-J 合并
