spacemacs使用小记

Nov 20, 2016 | Hits

写这篇博文主要是为了记录一些spacemacs的一些快捷键和一些使用方法，用于自己速查。

最近在fedora中安装了一次spacemacs，期间遇到了一些小问题，所以也一下。在安装的时候，提示找不到某些包或者某些地址访问不了的时候，可以在.spacemacs中将一个https的选项改为false，然后将最大等待时间改为30左右就可以了。

最近发现了一个小坑，发觉spacemacs的快捷键最近一直在换，有点不习惯

base

按键 说明
SPC b m kill other buffer
SPC b b show buffer list
SPC b s goto scratch
SPC b d kill this buffer
SPC w d kill this window

ssh

最近发现spacemacs一个超级棒的功能，在C-x C-f中输入url就可以使用tramp这个插件来访问远端的文件，可以像在本机一样进行操作，非常棒
url的格式如下/<protocal>:<username>@<serverIP>:<path>protocal一般为ssh

同时也支持跳转，

org-todo

在普通状态下:

按键 说明
t 讲项目设置为TODO项目或者在TODO，DONE，NONE之间进行切换
S-UP/DOWN 同上
, : 创建标签
M-right/left/up/down 调整当前标题的级别

VCS

按键 说明
SPC g i git init
SPC g s git status
c c git commit
s status a file
S status all files
