在html的布局中我们经常都要使用的position这个属性，下面是对position的一些总结：

常用的position属性及特点如下：

| 属性 | 特点 |
| ------ | ----- |
| absolute | 相对于于第一个非static属性的父元素进行定位，使用top，right等定位 |
| relative | 相对于父元素进行定位，但是只是视觉上，仍然占有位置，不脱离文档流，使用top，right等定位 |
| fixed | 相对于浏览器窗口进行定位，使用top，right等定位 |
| static | 默认使用的属性 |
| inherit | 继承父元素的定位属性 |

absolute 绝对定位

在absolute定位中，如果没有指定top，right等属性，它会自动选父元素进行定位，而不看父元素的定位属性。因此在不适用top等属性时，元素是没有脱离文档流的。

决定定位的元素可以通过z-index属性设计相互层叠关系，值得注意的是absolute元素之间的margin之间是不会发生重叠的

relative 相对定位

在相对定位中，无论是否使用top等属性，该元素都不会脱离文档流。

static 静态定位

默认定位方式，位于文档流中，top，right等属性不起作用
